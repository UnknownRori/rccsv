# RCCSV

Cross Platform(?) Helper library for manipulating `csv` file in C, and not very efficient so use it with caution.

## 🛠️ Development

```bash
# clone repository
> git clone https://github.com/UnknownRori/rccsv

# create static library
> make build-static

# run test
> make test
```
